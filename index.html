ImagePlayer

The image preview library.

This can be controlled with mouse (click, drag, scroll), touch (tap, swipe), and keyboard (tab and enter, arrows and escape).

Download: GitHub

Usage

1. Load the stylesheet and javascript.

HTML
<link rel="stylesheet" href="imageplayer.css">
<script src="imageplayer.js"></script>

2. Add the data-imageplayer attribute to the target a element.

HTML
<a href="image.jpg" data-imageplayer="default">Image</a>

3. Create an object of ImagePlayer class after loading the DOM.

JavaScript
window.addEventListener('DOMContentLoaded', () => new ImagePlayer());

Group Option

You can group by the value of data-imageplayer attribute.

HTML
<a href="image.jpg" data-imageplayer="Group Name">Image</a>

Comment Option

You can add comments with the data-header and data-footer attributes.

Comments can use \n to break lines.

HTML
<a href="image.jpg" data-imageplayer="Group Name" data-header="Header Comment" data-footer="Footer Comment">Image</a>

Example:

Blur Option

You can add a blur with the data-blur attribute.

Blur can toggled by clicking on the preview image.

HTML
<a href="image.jpg" data-imageplayer="Group Name" data-blur="true">Image</a>
