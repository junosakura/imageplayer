ImagePlayer

The image preview library. This can be controlled with mouse (click, drag, scroll), touch (tap, swipe), and keyboard (tab and enter, arrows and escape).

Download: GitHub

Usage

1. Load the stylesheet and javascript.

HTML
<link rel="stylesheet" href="imageplayer.css">
<script src="imageplayer.js"></script>

2. Add the data-imageplayer attribute to the target a element.

HTML
<a href="image.jpg" data-imageplayer="default">Image</a>

3. Create an object of ImagePlayer class after loading the DOM.

JavaScript
window.addEventListener('DOMContentLoaded', () => new ImagePlayer());

Attributes

1. You can group by the value of data-imageplayer attribute.

2. You can add comments with the data-header and data-footer attributes. Comments can use \n to break lines.

3. You can add a blur with the data-blur attribute. Blur can toggled by pressing the preview image.

HTML
<a href="image.jpg" data-imageplayer="Group Name" data-header="Header Comment" data-footer="Footer Comment" data-blur="true">Image</a>

Options

The following options can be added for each group:

1. Enable counter.

2. Enable loop.

3. Enable scrolling. If scrolling enabled, the target a element will scroll closer to the center of the screen.

4. Specify wait (animation speed). The default is 200.

JavaScript
window.addEventListener('DOMContentLoaded', () => {
const imageplayer = new ImagePlayer();
imageplayer.add({name:'Group Name A', counter:true, loop:true, scroll:true, wait:200});
imageplayer.add({name:'Group Name B', counter:true, loop:true, scroll:true, wait:400});
:
});
